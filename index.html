Trinity's Middle School Programming & Hardware Elective.

About: We Code Hardware

We Code Hardware (WCH) is a middle school elective that meets after school and covers both software and hardware. Students learn about the intersection of code and hardware through creating individual and group projects. Students work with:

  • Processing
  • Arduino Software
  • 123D Circuits
  • Arduino Uno
  • MaKey-MaKey
  • Paper Circuits
    • Circuit Stickers
  • LilyPad Arduino

Timeline:

Circuitry: Students are first introduced to circuitry through the MaKey-Makey. We use alligator clips to test how different connections can cause different actions on the laptop screen. We learn how humans are conductive by creating "Clappy Bird," interactive Flappy Bird. We high-five (clap) each other to cause the bird on the screen to jump between the pipes.

Paper Circuits: Students learned more about circuitry and interactive design through the use of circuit stickers to create their own holiday cards - they used LED stickers and different circuit schematics to create their own cards using different colored LEDs!

Arduinos: Students continued to learn more about hardware, specifically microcontrollers, by learning to work with the Arduino family of microcontrollers. Students created test programs such as the fundamental "Blink" program, and added their own twist by varying the time between each flash of the LED, using RGB LEDs, etc.

Wearables: Students built on this knowledge of Arduinos, and learned to create wearables with the Lilypad Arduino. Students uploaded and tested out various programs on the Lilypad board, and eventually wrote their own programs from scratch (not the programming language).

Programming in Processing: Students used the Processing programming language to create interactive games, equation solvers, and more. Students learned about conditional statements, loops, and other structures, ultimately integrating these tools into their projects.
